Tutorial: Pantalla de Inicio

Aprende a utilizar la pantalla de inicio de la aplicación. Haz scroll para ver cada sección explicada.

Vista General

Esta es la pantalla principal de inicio. A continuación te explicamos cada elemento destacado.

Iniciar Sesión

Haz clic en el botón Iniciar Sesión para acceder con tu usuario y contraseña.

Registro

Si aún no tienes cuenta, haz clic en Registrar para crear una nueva.

Accesibilidad

Haz clic en Accesibilidad para ajustar la aplicación a tus necesidades visuales en contraste o tamaño de letra.

Idioma

Puedes cambiar el idioma de la aplicación desde esta opción entre inglés o español.

Acerca de

Consulta información sobre la aplicación y sus desarrolladores.

Datos Curiosos

Descubre datos interesantes y curiosidades relacionadas con la aplicación y el mundo del reciclaje cada 30 segundos.

Cerrar

Haz clic aquí para cerrar la aplicación de forma segura.

Minimizar

Puedes minimizar la ventana de la aplicación con este botón.
